# Examples

Real apps built with rooted, kept in the monorepo.

- [`recipe-book/`](./recipe-book) is the canonical example app. It is the user-facing reference for the vertical-slice layout described in [`docs/guide/application-model.md`](../docs/guide/application-model.md), and it is also the regression surface where every shipped framework feature is exercised end-to-end. When something breaks here, releases halt.

If you're new to rooted, read [`docs/guide/getting-started.md`](../docs/guide/getting-started.md) first, then open the recipe-book to see the same ideas at scale.
