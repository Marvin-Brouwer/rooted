# [`@rooted/elements`](https://www.npmjs.com/package/@rooted/elements)

Html element utilities for the  [@rooted/*](https://github.com/Marvin-Brouwer/rooted?tab=readme-ov-file#rooted) framework.

This library creates `HTMLElement`s and can also be used stand-alone.

> [!IMPORTANT] This package is still in `ALPHA`
