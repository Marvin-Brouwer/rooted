# [`@rooted/util`](https://www.npmjs.com/package/@rooted/util)

Internal utilities for the [`@rooted/*`](https://github.com/Marvin-Brouwer/rooted#rooted) framework. Not part of the public API.

> [!IMPORTANT]
> This package is still in alpha.

The exports here are consumed by other `@rooted/*` packages. They are not stable and may change without a release-note entry. App code should not depend on this package directly.
