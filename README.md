# rooted

> Web components rooted very close to the original.

Rooted is a minimal, framework-free TypeScript library for building web applications with native
[custom elements](https://developer.mozilla.org/en-US/docs/Web/API/Web_components/Using_custom_elements).
It provides a thin, type-safe layer over the browser's own component model — no virtual DOM,
no JSX, no magic compilation step beyond standard TypeScript.

```ts
import styles from './greeting.css'
import { component } from '@rooted/components'

export const Greeting = component({
  name: 'greeting',
  styles,
  onMount({ append }) {
    append('p', { textContent: 'Hello from rooted!' })
  },
})
```

## Installation

Install only what you need:

```sh
# Component system
pnpm add @rooted/components

# Client-side router (requires @rooted/components)
pnpm add @rooted/router

# Vite plugin for automatic route discovery (dev dependency)
pnpm add -D @rooted/router
```

## Quick start

### 1. Create a component

```ts
// src/counter.mts
import styles from './counter.css'
import { component } from '@rooted/components'

export const Counter = component({
  name: 'counter',
  styles,
  onMount({ append, signal }) {
    let count = 0

    const label = append('span', { textContent: '0' })

    const btn = append('button', { textContent: 'Increment' })
    btn.addEventListener('click', () => {
      count++
      label.textContent = String(count)
    }, { signal })
    // ↑ listener is removed automatically when the component unmounts
  },
})
```

### 2. Bootstrap the application

```ts
// src/main.mts
import { application } from '@rooted/components/application'
import { component } from '@rooted/components'
import { Counter } from './counter.mts'

const App = component({
  name: 'app',
  onMount({ append }) {
    append('h1', { textContent: 'My App' })
    append(Counter)
  },
})

application(App) // mounts to #app by default
```

### 3. Add routing (optional)

```ts
// src/routes/article/_gates.mts
import { gate, token } from '@rooted/router'
import { Article } from './article.mts'

export const ArticleGate = gate(Article)`/articles/${token('id', Number)}/`
```

```ts
// src/main.mts
import { application } from '@rooted/components/application'
import { router } from '@rooted/router'
import * as appRoutes from './_routes.g.mts' // generated by Vite plugin

const Router = router({
  home:     HomeComponent,
  notFound: NotFoundComponent,
  ...appRoutes,
})

application(component({
  name: 'app',
  onMount({ append }) { append(Router) },
}))
```

## Browser support

Rooted uses `@scope` for CSS isolation, which requires:

| Browser | Minimum version |
|---------|----------------|
| Chrome  | 118            |
| Firefox | 128            |
| Safari  | 17.4           |

On older browsers the CSS nesting fallback is used, which may allow minor style
leakage in deeply nested scenarios.

## Development

```sh
# Clone and install
git clone https://github.com/Marvin-Brouwer/rooted.git
cd rooted
pnpm install

# Run all packages in watch mode
pnpm dev:all

# Run the example app
cd example && pnpm dev

# Run tests
pnpm test

# Run tests in watch mode
pnpm test:watch

# Lint (auto-fix)
pnpm lint

# Build all packages
pnpm build:all
```
